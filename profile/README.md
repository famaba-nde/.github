# famaba-nde

This organization contains all repositories relating to the internship project by Liam Van Rie at E-Crane/E-Towers for NDE data.

It has 5 main interlinked components:
* Frontend (viewer frontend, vanillaJS)
* Backend (viewer backend, python with FastAPI)
* Processor (nde file processing framework, python)
* Parsing (nde file parsing, python)
* Documentation (common documentation, markdown)

As well as a handful of deprecated repos or testing environments.
